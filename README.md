# 在线学习平台

## 项目介绍

一个基于Django后端和Vue前端的在线学习平台，包括教师、学生、管理员三种用户角色，主要功能包括：

- 教师可以创建和管理课程，包括视频、作业和学习资料
- 学生可以观看课程视频、提交作业和下载资料
- 系统自动跟踪学习进度和成绩并生成统计报表
- 管理员可以对所有数据进行编辑

## 技术架构

### 后端技术
- Django 3.2 + Django REST framework
- MySQL 数据库
- RESTful API 接口设计

### 前端技术
- Vue 3 框架
- Element Plus UI组件库
- Axios 网络请求
- ECharts 数据可视化
